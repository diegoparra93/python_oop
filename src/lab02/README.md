# Лабораторная работа №2 - Коллекция объектов

## Класс UserCollection

### Реализованные методы

| Метод | Описание |
|------|----------|
| add(user) | Добавить пользователя |
| remove(user) | Удалить пользователя |
| get_all() | Получить всех пользователей |
| find_by_email(email) | Поиск по email |
| find_by_city(city) | Поиск по городу |
| remove_at(index) | Удаление по индексу |
| sort_by_name() | Сортировка по имени |
| get_adults() | Получить только совершеннолетних пользователей |

---

### Магические методы

- __len__() — количество пользователей  
- __iter__() — перебор в цикле for  
- __getitem__() — доступ по индексу  

---

## Сценарий 1: Базовые операции

**Что демонстрирует:**  
создание коллекции, добавление и удаление пользователей, работу __len__ и __iter__.

**Как работает:**  
Создается коллекция пользователей. В неё добавляются несколько пользователей.  
С помощью len() выводится количество пользователей.  
Через цикл for перебираются все пользователи.  
Затем один пользователь удаляется, и коллекция выводится снова.

## Сценарий 2: Поиск и ограничения

**Что демонстрирует:**  
работу поиска и проверку ограничений.

**Как работает:**  
Создается коллекция с пользователями.  
Выполняется поиск пользователя по email и по городу.  
Также проверяется защита:
- нельзя добавить объект не типа User  
- нельзя добавить дубликат пользователя (по email)

## Сценарий 3: Индексация и сортировка

**Что демонстрирует:**  
доступ по индексу, удаление, сортировку и фильтрацию.

**Как работает:**  
Создается коллекция пользователей.  
Доступ к элементам осуществляется по индексу.  
Удаляется элемент методом remove_at().  
Коллекция сортируется по имени.  
Метод get_adults() возвращает только совершеннолетних пользователей.
